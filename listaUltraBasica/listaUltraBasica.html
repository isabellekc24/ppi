a) O que é Internet e WEB? São as mesmas coisas? Explique de forma clara e evidencie as diferenças.

A Internet é uma rede global que conecta dispositivos e permite a troca de dados. A Web é um serviço que funciona na Internet, permitindo o acesso a páginas e recursos via navegadores. Assim, a Internet é a base, e a Web é um dos serviços que utilizamos nela.

b) O que é arquitetura cliente-servidor utilizada na web ? Explique o carregamento de uma página passo a passo.

Na arquitetura cliente-servidor, o navegador solicita uma página ao servidor, que a processa e envia de volta. O navegador então exibe a página para o usuário.

c) O que é um tag HTML?

Uma tag HTML define elementos em uma página web, sendo delimitada por < > e usada para estruturar o conteúdo.

d) O que é HTML e qual a estrutura básica de um documento HTML?

É a linguagem de marcação usada para criar e estruturar páginas da web. A estrutura básica de um documento HTML inclui as tags DOCTYPE, html, head e body, onde head contém informações sobre a página e body abriga o conteúdo visível.

            <!DOCTYPE html>
            <html lang="pt-BR">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Título da Página</title>
            </head>
            <body>
                <h1>Título Principal</h1>
                <p>Este é um parágrafo de exemplo.</p>
            </body>
            </html>
